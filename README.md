**[披风站](http://121.37.27.192:233/capes.html) ** 它的优点包括

- **轻量**, 小巧轻便，披风的功能仅用了6个类
- **版本支持**, 目前为数不多的使用fabric创建的mod，支持1.12-1.16(快照)
- **自定义性**, 您可以选择官方各种颜色的火云邪神披风/旗帜披风/Minecon披风(对本mod或相关产品有贡献的用户会获得赠送的披风)
